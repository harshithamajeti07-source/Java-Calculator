# Smart Calculator

A fully functional desktop calculator built with Java Swing, featuring basic arithmetic operations (+, -, *, /), decimal support, and a clean, intuitive interface.



## ✨ Features

- **Basic Arithmetic Operations**: Addition, subtraction, multiplication, and division
- **Decimal Support**: Handle decimal numbers seamlessly
- **Clear Display**: Reset calculator with 'C' button
- **Responsive UI**: Grid-based layout mimicking standard calculators
- **Professional Styling**: Arial font with bold numbers and light gray operation buttons

## 🛠️ Tech Stack

- **Language**: Java
- **GUI Framework**: Java Swing (JFrame, JPanel, JButton, JTextField)
- **Layout**: GridLayout for organized button arrangement
- **Event Handling**: ActionListener interface
